# 百大科创 H5 —— "AI奇妙之旅：从魔法到科学"

## 项目结构

```
百大科创H5/
├── index.html          ← 主页面（双击打开即可预览）
├── css/
│   └── style.css       ← 全部样式
├── js/
│   ├── data.js         ← 题库/时间轴/对话数据
│   └── main.js         ← 交互逻辑
├── images/             ← 图片素材（可后续添加）
└── README.md           ← 本文件
```

## 本地预览

直接双击 `index.html` 在浏览器中打开。

## 部署到线上（获取在线链接）

### 方式一：GitHub Pages（免费，推荐）
```bash
# 1. 在 GitHub 创建仓库
# 2. 上传整个百大科创H5文件夹
# 3. Settings → Pages → Source: main branch → Save
# 4. 获得链接：https://你的用户名.github.io/仓库名/
```

### 方式二：Vercel（免费）
```bash
# 直接拖拽文件夹到 vercel.com 即可部署
```

### 方式三：Netlify（免费）
```bash
# 直接拖拽文件夹到 netlify.com 即可部署
```
